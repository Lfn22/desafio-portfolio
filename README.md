## 📋 Sobre o projeto

Primeiro projeto de portfólio desenvolvido como desafio prático durante o curso de **HTML e CSS da Alura**. Foco em estrutura semântica, estilização com CSS e boas práticas de desenvolvimento front-end.

---

## ✨ Conteúdo

- Página de apresentação profissional
- Seção de habilidades e tecnologias
- Links de contato e redes sociais
- Layout responsivo

---

## 🚀 Como rodar localmente

```bash
# 1. Clone o repositório
git clone https://github.com/Lfn22/desafio-portfolio.git

# 2. Entre na pasta
cd desafio-portfolio

# 3. Abra no navegador
open index.html
```
